controlListener(self)

source code 

The control listener. Listens for the STOPSANITYSERVICE commands, which shuts down the service by cancelling the timers and stopping the Manual Sanity Check Request Listener.

checkConfig(self)

source code 

The configuration check thread. Checks the database every minute to see if the intervals have changed. Sets the class variables accordingly.

NormalCheck(self)

source code 

The automatic normal sanity check. Normal sanity checking involves the Mainframe to check every port used by the modules on all the team's VMs. When it can't connect to a specific port it adds an entry to the suspicious IP table.

P2PCheck(self)

source code 

The automatic Peer-to-Peer sanity check thread. Peer-to-peer sanity checking involves the mainframe asking all the VMs to scan each other. When a VM finds a suspicious client the IP of the reporting VM will also be added to the database.
